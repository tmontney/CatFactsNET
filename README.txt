Built using Visual Studio 2017. Open using VS2017, switch to Release mode, build.

Special characters may appear in browser as "?", such as when trying to display the British pound sign. Run the exe as admin, call 127.0.0.1:1337/API/CATFACT. Data returns as raw string.

IP and port can be changed in Core.vb. EXE can be run as regular user via netsh http add urlacl command.
